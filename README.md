🧠 AI Models for Mental Health Detection
🚀 Overview

This project aims to detect early mental health indicators from social media text using machine learning and NLP techniques.

🎯 Problem

Mental health issues often go unnoticed in early stages.
This project explores how NLP can help identify signals from text data to support early intervention.

⚙️ Approach
Preprocessed social media text data
Trained and compared multiple models:
Logistic Regression
Support Vector Machine (SVM)
BERT (transformer-based model)
Evaluated using:
F1-score
ROC-AUC
💻 Features
Real-time text input analysis
Side-by-side model comparison
Interactive Flask dashboard
